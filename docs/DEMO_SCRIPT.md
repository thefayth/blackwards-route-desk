# Demo Script

Target duration: 2 minutes 30 seconds. AI narration is disclosed in the YouTube description.

## 0:00-0:18 — Problem

Black creators are often asked to move fast while ownership, credit, compensation, and audience access stay vague. Auntie AI reads the route before the work moves.

## 0:18-0:48 — Immediate Product

No account and no upload. Choose the kind of route, describe only the public-safe visible terms, and mark compensation, rights, credit, audience access, and urgency.

## 0:48-1:22 — GPT-5.6 Packet

Submit one synthetic opportunity. Show the verdict, route reading, ownership checks, extraction signals, questions before yes, 24-hour action, and receipt checklist. Copy the Markdown packet.

## 1:22-1:48 — Resilient Samples

Open the worldwide exhibition and accelerator cases. Explain that the three verified packets remain available without spending a model run and are always labeled as samples.

## 1:48-2:10 — Implementation

GPT-5.6 uses strict structured output and `store: false`. D1 records hashes, counters, model status, and latency, never the creator's summary or packet. Private-contact and credential patterns are rejected before generation.

## 2:10-2:30 — Codex And Impact

Codex separated this contest build from the production Black2Africa platform, implemented and tested the full app, and prepared the evidence trail. The result is one small decision tool with a larger rule: ownership before movement.
